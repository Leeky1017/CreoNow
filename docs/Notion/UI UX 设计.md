# 🎨 UI/UX 设计

## 设计原则

<aside>
🎯

CN 的设计应该让创作者 **专注于创作本身**，而不是与工具搏斗。

</aside>

- **简洁优先**：界面干净，减少干扰，让文字成为主角
- **渐进式复杂度**：基础功能开箱即用，高级功能按需展开
- **快速反馈**：AI 操作的响应要即时可见
- **可回退**：所有 AI 修改都可以轻松撤销

---

## 核心界面区域

### 1. 编辑区（Editor）

- TipTap 富文本编辑器
- 所见即所得的写作体验
- Inline AI 操作（选中文字 → 技能菜单）

### 2. 侧边栏（Sidebar）

- 项目文件树
- 知识图谱浏览
- 角色 / 地点 / 事件快速访问

### 3. AI 面板（AI Panel）

- 对话式交互
- 技能触发与预览
- 上下文状态可视化

### 4. 版本面板（Version Panel）

- 版本时间线
- Diff 对比视图
- 快速恢复操作
