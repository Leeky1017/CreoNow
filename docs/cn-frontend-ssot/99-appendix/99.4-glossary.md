# 99.4 术语表（Glossary）

> 本术语表收录 CreoNow 前端文档中反复出现的核心概念，供快速查阅。

| 术语 | 定义 |
|------|------|
| **SSOT** | Single Source of Truth（唯一真源）。`docs/cn-frontend-ssot/` 是 CreoNow 前端的唯一权威文档来源 |
| **Design Token** | 以 CSS 自定义属性（`--color-*`、`--space-*` 等）定义的设计原子值，是颜色/间距/动画等视觉参数的唯一存取接口 |
| **Primitive** | 最小粒度的 UI 原子组件（Button、Input、Select 等），不含业务逻辑，位于 `primitives/` 目录 |
| **Composite** | 由多个 Primitive 组合而成的复合组件（SearchBar、FileNode 等），位于 `composites/` 目录 |
| **Feature Component** | 绑定特定业务逻辑与 IPC 调用的功能组件（AIPanel、EditorView 等），位于 `features/` 目录 |
| **Shell** | App Shell，应用的骨架层（`AppLayout`），承载路由出口、布局状态、全局浮层挂载 |
| **IPC Bridge** | Electron 的 `contextBridge` 跨进程通信桥，是渲染进程与主进程通信的唯一合法通道 |
| **Guard Test** | 守卫测试，用于验证跨层约束（如 IPC 契约对齐、Design Token 覆盖率等）不被意外破坏 |
| **Contract Test** | 契约测试，验证 IPC 通道的请求/响应类型在主进程与渲染进程之间保持一致 |
| **Zen Mode** | 全屏沉浸写作模式，隐藏所有 UI chrome（Icon Rail、Context Panel、AI FAB），仅保留编辑器 |
| **Focus Mode** | AI 聚焦模式，编辑器与 AI 面板 50:50 分栏，其余 UI 隐藏 |
| **Cinematic Calm** | CreoNow 的核心设计哲学——界面为创作让路、AI 为创作赋能，一切视觉决策服务于「安静而专注」的创作氛围 |
| **Accent Color** | 产品的全局强调色 `#7AA2F7`，统一用于所有激活/选中/聚焦/交互反馈状态 |
