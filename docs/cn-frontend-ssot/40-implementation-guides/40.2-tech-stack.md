## 核心技术栈

| **层** | **技术** | **版本** | **选型理由** |
| --- | --- | --- | --- |
| 框架 | **React** | ^18.3 | 生态最大，Electron 官推，AI 生成质量最高 |
| 状态 | **Zustand** | ^5.0 | 零 boilerplate，selector 隔离渲染，persist 中间件即插即用 |
| 编辑器 | **TipTap 2**（ProseMirror） | ^2.10 | 中文 IME 最成熟，扩展机制比 Slate 更可预测 |
| 样式 | **Tailwind CSS** | ^4.0 | 与 Design Token 变量无缝集成，JIT 编译零冗余 |
| 组件 | **Radix UI**（via shadcn/ui） | latest | 无样式、WAI-ARIA 完整、focus-trap 内置 |
| 图标 | **Lucide React** | ^0.470 | MIT 协议，tree-shakable，风格统一 |
| 拖拽 | **@dnd-kit** | ^6.0 | 文件树排序、面板拖拽停靠 |
| 测试 | **Vitest** + **@testing-library/react** | Vitest ^3.0 · RTL ^16.0 | 与 Vite 生态一致，组件测试标准方案 |
