# 20.2 组件规范

> 来源：[docs/CN-设计文稿.md](../../CN-设计文稿.md) §3–§8、§11–§13

本文件收录各功能组件的 V3 视觉规范。每个组件的状态矩阵详见 [20.3 组件状态](20.3-component-states.md)。

## Icon Rail + Context Panel（取自 V2，视觉微调）

### Icon Rail 规范

| **属性** | **V2 原值** | **V3 调整** |
| --- | --- | --- |
| 宽度 | 48px | 48px |
| 背景色 | `#0A0A0A` | `#000000`（回退 V1 纯黑） |
| 图标尺寸 | 20×20px stroke 1.5px | 18×18px stroke 1.5px |
| 可点击区域 | 36×36px | 36×36px |
| 图标间距 | 4px | 6px |
| Hover 背景 | `#1E1E1E` | `rgba(255,255,255,0.06)` |
| 激活色 | `#7AA2F7` | `#7AA2F7` |
| Hover 动画 | 无指定 | `ease-out 150ms` |

图标列表（与 V2 一致）：上区 Search → LayoutDashboard → Folder → BarChart2 → Calendar · 分隔线 · 下区 User → Network · flex-1 留白 · 底部 Settings

### Context Panel 调整

- 背景色：`#000000`（V1）
- 文件树节点高度：32px（V2）
- 选中态：`--accent-subtle` 背景 + 左侧 2px `--accent` 指示线
- 大纲激活色：`#7AA2F7`
- 字数统计底栏：保留 V2 设计

## AI 面板（Cursor 式，§4）

整体布局：顶栏（New Chat + 历史）→ 对话区域（可滚动）→ 输入框 → 底部工具栏（三个弹出式选择器）

### 底部工具栏三个选择器

| **CN 模式** | **对应 Cursor** | **说明** |
| --- | --- | --- |
| Agent | Agent | 自主执行：续写、修改编辑器内容 |
| Plan | Plan | 先给计划，用户确认后执行 |
| Analyze | Debug 变体 | 分析模式：只读，不修改编辑器。IPC 映射为 `mode: "ask"` + 分析类 skillId |
| Ask | Ask | 纯对话 |

### 对话区域规范

| **元素** | **规范** |
| --- | --- |
| 用户消息 | 左对齐，背景 `rgba(255,255,255,0.04)` 圆角 8px padding 12px，Inter 13px `#E8E8E8` |
| Creo 回复（文学内容） | Source Serif 4 **15px** `#E8E8E8`，行高 1.8。Focus Mode 下改为 16px |
| Creo 回复（分析/建议） | Inter 13px `#B0B0B0`，行高 1.6 |
| 行动栏 | 接受 · 编辑 · 重新生成 · 复制。12px `#555555`，hover `#F0F0F0` |

## AI FAB 交互规范（§5）

| **手势** | **V3 行为** | **时序** |
| --- | --- | --- |
| 单击 | 打开/关闭 AI Sidebar | 延迟 300ms |
| 双击 | 展开快速菜单 | 第二击立即执行 |
| 长按 500ms | 进入 AI Focus 模式 | 松手执行 |
| 单击（有选中文字时） | 弹出 Inline 工具栏 | 无延迟 |

FAB 视觉：44×44px 圆形，`#7AA2F7`，Lucide `Bot` 20px，`--z-fab`（100）

## AI Focus Mode（§6）

左编辑器 50% + 右 AI 面板 50%，Icon Rail / Context Panel / FAB 全隐藏。三入口：FAB 长按 / `Ctrl+Shift+F` / AI 面板 Maximize 图标。

## 搜索面板 ⌘K（§7）

V1 的左右分栏 + 预览面板，加 V2 的 motion 动画。5 类搜索：Document / Project / Character / Task / **AI Actions**（新增）。

## 编辑器（§8）

| **属性** | **V3 值** |
| --- | --- |
| 背景色 | `#050505` |
| 正文字体 | Source Serif 4 16px |
| 行高 | 1.8 |
| 左右 padding | 80px |
| Zen Mode 最大宽度 | 640px |

AI Inline Block：无容器，仅左侧 2px `#7AA2F7` 指示线 + padding-left 12px。AI 内容字色 `#C0C0C0`。

## Dashboard（§11）

页面结构：迎接区 → KPI 卡片区（4 列）→ Heatmap（52×7）→ 最近文档 + AI 建议（6:4 双栏）→ 底部快速记录。

Heatmap 色阶：`#0A0A0A` → `rgba(122,162,247,0.15)` → `rgba(122,162,247,0.35)` → `rgba(122,162,247,0.60)` → `#7AA2F7`

## Settings Modal（§12）

720px 宽，最大 85vh，5 Tab：外观 · 编辑器 · AI 与 Creo · 账户 · 快捷键。Tab 激活态：左侧 2px `--accent` 指示线 + `--accent-subtle` 背景。

## 路由结构（§13）

```
/app/dashboard        ← 创作指挥中心
/app/projects         ← 项目列表
/app/files            ← 文件浏览器
/app/editor/:docId    ← 编辑器
/app/analytics        ← 数据分析
/app/characters       ← 角色管理
/app/knowledge-graph  ← 知识图谱
/app/calendar         ← 日历
/onboarding           ← 首次启动引导
```
