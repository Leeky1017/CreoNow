# 20.5 间距系统

> 来源：[docs/CN-设计文稿.md](../../CN-设计文稿.md) §23（仅 23.1–23.4 间距内容）

**基础单位：4px。所有间距必须是 4 的倍数。** 开发不得使用系统外的间距值（如 5px、10px、15px、18px）。

## 间距档位表

| **Token** | **值** | **Tailwind** | **语义** | **典型用途** |
| --- | --- | --- | --- | --- |
| `--space-1` | 4px | `p-1` | 最小间距 | 图标与文字间距、Badge 内边距 |
| `--space-2` | 8px | `p-2` | 紧凑间距 | 列表项 padding、按钮内图标间距 |
| `--space-3` | 12px | `p-3` | 标准小间距 | 对话气泡 padding、卡片标题与内容间距 |
| `--space-4` | 16px | `p-4` | 标准间距 | 面板 padding、列表分组间距 |
| `--space-6` | 24px | `p-6` | 宽松间距 | 卡片之间、Settings Section 间距 |
| `--space-8` | 32px | `p-8` | 大间距 | Modal padding、页面级 Section 间距 |
| `--space-12` | 48px | `p-12` | 超大间距 | 空状态图示与文字间距 |
| `--space-16` | 64px | `p-16` | 最大间距 | 编辑器左右 padding |

## 组件内部间距规范

| **组件** | **内部 padding** | **元素间距（gap）** |
| --- | --- | --- |
| Icon Rail 图标按钮 | 可点击 36×36px，图标 18×18px 居中 | 图标间 6px |
| Context Panel 文件节点 | 左右 12px，上下 8px | 图标与文字 8px |
| AI 面板对话气泡 | 全向 12px | 消息之间 16px |
| AI 面板输入框 | 左右 12px，上下 12px | 与底部工具栏 8px |
| 底部工具栏 | 左右 12px，上下 8px | 选择器间 8px |
| Popover 列表行 | 左右 12px，上下 8px | 行高 36px |
| 搜索面板搜索框 | 左右 16px | 图标与输入框 12px |
| 编辑器正文区 | 左右 80px；顶部 32px | 段落间 16px；标题与正文 12px |
| Dashboard 卡片 | 全向 24px | 卡片间 24px |
| Settings Modal | 内容区 32px | Tab 与内容 24px；表单项间 16px |

## 间距节奏原则

- **相关元素靠近，不相关元素拉开**——同组 `--space-2/3`，不同组 `--space-6/8`
- **垂直间距 > 水平间距**——符合阅读习惯
- **不要用 padding 和 margin 混用修补视觉问题**——从 Token 根源解决
- **编辑区排版间距独立**——由 `line-height` 和 `margin-bottom` 控制

## CSS Token 声明

```css
:root {
  --space-1:   4px;
  --space-2:   8px;
  --space-3:  12px;
  --space-4:  16px;
  --space-6:  24px;
  --space-8:  32px;
  --space-12: 48px;
  --space-16: 64px;

  /* Semantic aliases */
  --space-icon-gap:       var(--space-2);
  --space-item-padding-x: var(--space-3);
  --space-item-padding-y: var(--space-2);
  --space-panel-padding:  var(--space-4);
  --space-card-padding:   var(--space-6);
  --space-section-gap:    var(--space-8);
}
```
