# 20.4 动画系统

> 来源：[docs/CN-设计文稿.md](../../CN-设计文稿.md) §20

**设计原则：动画是功能，不是装饰。** 动画的存在必须有理由：引导视线、表达状态变化、建立空间感。所有 duration 和 easing 必须使用系统 Token。

## Duration 分级

| **Token** | **值** | **适用场景** | **不适用** |
| --- | --- | --- | --- |
| `--duration-instant` | 80ms | 按下反馈、颜色切换 | 面板展开 |
| `--duration-fast` | 150ms | Tooltip 显示、Popover 收起、图标颜色变化 | 大面积布局变化 |
| `--duration-normal` | 200ms | Popover 展开、Modal 进入、面板切换 | 页面路由切换 |
| `--duration-slow` | 300ms | 页面路由过渡、Context Panel 展开/收起、Zen Mode | 微交互 |
| `--duration-veryslow` | 500ms | 初次加载 Splash、KG 节点入场 stagger | 重复触发动画 |

## Easing 曲线

| **Token** | **CSS 值** | **适用场景** |
| --- | --- | --- |
| `--ease-out` | `cubic-bezier(0.0, 0.0, 0.2, 1.0)` | 元素进入（首选） |
| `--ease-in` | `cubic-bezier(0.4, 0.0, 1.0, 1.0)` | 元素退出（首选） |
| `--ease-in-out` | `cubic-bezier(0.4, 0.0, 0.2, 1.0)` | 颜色、opacity 变化 |
| `--ease-spring` | `cubic-bezier(0.34, 1.56, 0.64, 1.0)` | FAB hover、快速菜单弹出 |
| `--ease-linear` | `linear` | Spinner 旋转、Progress Bar |

## 各组件动画规格

| **组件** | **进入** | **退出** |
| --- | --- | --- |
| Context Panel | `translateX(-100%)→0`，slow，ease-out | 反向，slow，ease-in |
| AI 面板 | `translateX(100%)→0`，slow，ease-out | 反向，slow，ease-in |
| 搜索面板 | `opacity 0→1` + `scale(0.97)→1`，slow，ease-out | `opacity 1→0` + `scale→0.97`，fast，ease-in |
| Popover | `opacity 0→1` + `translateY(-4px)→0`，normal，ease-out | 反向，fast，ease-in |
| FAB 快速菜单 | 弧形展开，stagger 40ms，ease-spring | 反向 stagger 20ms，fast，ease-in |
| Zen Mode | Shell 元素各自 translateX + opacity→0，stagger 0/50/50/80ms | 反向 |
| AI Inline Block | 每字符 opacity 0→1，30ms，模拟书写 | 指示线 opacity→0，fast |
| KG 节点 | `scale(0)→1`，stagger 40ms，ease-spring | `opacity→0`，fast |
| Toast | `translateY(100%)→0`，normal，ease-out | `translateY→100%` + `opacity→0`，fast，ease-in |
| 路由切换 | 进入页 `opacity 0→1`，slow | 退出先完成（100ms），进入再开始 |

Toast 自动消失：成功 3s，警告 5s，错误 8s 或手动关闭。

## 禁用动画的场景

- `prefers-reduced-motion: reduce`——所有动画缩短为 1ms
- 数据实时更新（字数每次击键）——直接切换
- 流式文字输出——字符级已有动画，外容器不加 fade-in
- Tooltip——退出用 `--duration-instant`

## CSS Token 声明

```css
:root {
  --duration-instant:  80ms;
  --duration-fast:    150ms;
  --duration-normal:  200ms;
  --duration-slow:    300ms;
  --duration-veryslow:500ms;

  --ease-out:     cubic-bezier(0.0,  0.0,  0.2,  1.0);
  --ease-in:      cubic-bezier(0.4,  0.0,  1.0,  1.0);
  --ease-in-out:  cubic-bezier(0.4,  0.0,  0.2,  1.0);
  --ease-spring:  cubic-bezier(0.34, 1.56, 0.64, 1.0);
  --ease-linear:  linear;
  --stagger-base: 40ms;
}

@media (prefers-reduced-motion: reduce) {
  :root {
    --duration-instant:   1ms;
    --duration-fast:      1ms;
    --duration-normal:    1ms;
    --duration-slow:      1ms;
    --duration-veryslow:  1ms;
  }
}
```
