# 20.1 色彩系统

> 来源：[docs/CN-设计文稿.md](../../CN-设计文稿.md) §2、§9、§24、§21

## 核心哲学

**一个 accent，贯穿全局。** V3 的 `#7AA2F7` 是产品的统一 accent 色，用于所有激活态与交互态。AI 面板的 Tab、Icon Rail 的激活图标、按钮 hover、选中的文件节点——都用同一个 accent。一致性 = 融合感。

## 基础色板

| **Token** | **V1** | **V2** | **V3** | **理由** |
| --- | --- | --- | --- | --- |
| `--background` | `#050505` | `#0D0D0D` | `#050505` | 回退 V1——更深的黑底给文字更强的浮雕感 |
| `--card` | `#0A0A0A` | `#111111` | `#0A0A0A` | 跟随背景回退 |
| `--muted` | `#1A1A1A` | `#1E1E1E` | `#1A1A1A` | 跟随背景回退 |
| `--accent` | `#1A1A1A`（缺失） | `#7AA2F7` | `#7AA2F7` | V2 方向正确，V3 继承并落实为全局 accent |
| `--accent-foreground` | — | `#050505` | `#050505` | accent 底色上的文字用深色 |
| `--accent-hover` | — | — | `#8BB3F8` | 比 accent 亮 ~8% |
| `--accent-subtle` | — | — | `rgba(122,162,247,0.08)` | 选中行背景、活动节点底色 |
| `--accent-muted` | — | — | `rgba(122,162,247,0.20)` | 边框、分隔线等中等饱和场景 |
| `--sidebar` | `#000000` | `#0A0A0A` | `#000000` | 回退 V1 纯黑——侧边栏应该「消失」在背景中 |
| `--sidebar-primary` | `#FFFFFF` | `#7AA2F7` | `#7AA2F7` | 激活图标用产品 accent |

## Accent 色使用规则（全局一致）

凡是「激活/选中/聚焦/交互反馈」，一律用 `#7AA2F7`。区分 AI 内容与用户内容靠的是**图标（Lucide Sparkles）和排版**，而非另起一套颜色。

| **场景** | **使用方式** | **Token** |
| --- | --- | --- |
| Icon Rail 激活图标 | 图标本身染色 `#7AA2F7` | `--accent` |
| Context Panel 文件选中行 | 左侧 2px 指示线 + 底色 | `--accent` / `--accent-subtle` |
| 编辑器工具栏激活按钮 | 按钮底色 + 文字 | `--accent-subtle` / `--accent` |
| AI 面板 Tab 激活指示器 | 底部 2px 线——与其他 Tab 完全相同 | `--accent` |
| AI Inline Block 左侧指示线 | 左侧 2px（标识 AI 内容） | `--accent` |
| AI FAB | 按钮背景 accent | `--accent` |
| 选择器当前选中项 | 文字 accent，行底色 subtle | `--accent` / `--accent-subtle` |
| Settings Tab 激活态 | 底部 2px 线 | `--accent` |
| 搜索面板选中结果行 | 底色 | `--accent-subtle` |
| Dashboard KPI 数字 | 保持 `--foreground`（`#F0F0F0`） | `--foreground` |
| Heatmap 色阶 | accent 渐变 | `--accent` 系列 |
| 输入框 focus ring | `--accent-muted` 边框 | `--accent-muted` |

## 视觉层级（Elevation）

深色界面的层级靠「越亮越近」。背景最深，往上每升一层背景色略微变亮。

| **层级** | **名称** | **背景色** | **Token** | **组件** |
| --- | --- | --- | --- | --- |
| Layer 0 | 地基（Base） | `#050505` | `--background` | App 最底层背景，编辑器正文区 |
| Layer 1 | 侧边栏（Rail） | `#000000` | `--sidebar` | Icon Rail、Context Panel、AI 面板 |
| Layer 2 | 卡片/面板（Surface） | `#0A0A0A` | `--card` | Dashboard 卡片、AI 对话气泡 |
| Layer 3 | 悬浮元素（Raised） | `#111111` | `--raised` | Hover 浮起卡片、展开的 Accordion |
| Layer 4 | 浮层（Overlay） | `#161616` | `--overlay` | Popover、Dropdown、Context Menu |
| Layer 5 | 模态（Modal） | `#1A1A1A` | `--modal` | Settings Modal、确认对话框 |
| Layer 6 | 最顶层（Toast） | `#222222` | `--toast` | Toast 通知、全局 Spinner |

### 阴影体系

| **Token** | **CSS 值** | **适用层级** |
| --- | --- | --- |
| `--shadow-sm` | `0 1px 3px rgba(0,0,0,0.4)` | Layer 3 |
| `--shadow-md` | `0 4px 16px rgba(0,0,0,0.5)` | Layer 4 |
| `--shadow-lg` | `0 8px 32px rgba(0,0,0,0.6)` | Layer 5 |
| `--shadow-xl` | `0 16px 48px rgba(0,0,0,0.7)` | Layer 6 |
| `--shadow-accent` | `0 0 0 6px rgba(122,162,247,0.15)` | FAB hover 专用 |
