## 验收流程

```
1. 确认所有涉及的单个组件已通过验收
2. 创建组合场景的 Story 或测试页面
3. AI 可视化自检组合效果
4. 用户人工验收整体体验
5. 记录问题并修复
6. 更新场景状态为"已验收"
```
