# Editor Specification Delta

## Change: s3-zen-mode

### Requirement: Editor 必须支持禅模式沉浸式写作切换 [ADDED]

编辑器必须支持禅模式开关：进入禅模式后隐藏侧边栏/面板并突出主编辑区，退出后可恢复原状态。

#### Scenario: S3-ZEN-MODE-S1 进入禅模式隐藏干扰面板 [ADDED]

- **假设** 用户在正常编辑布局中
- **当** 用户触发进入禅模式
- **则** 左侧栏与右侧面板隐藏
- **并且** 主编辑区扩展为主要可视区域

#### Scenario: S3-ZEN-MODE-S2 退出禅模式恢复进入前布局 [ADDED]

- **假设** 用户已在禅模式中编辑
- **当** 用户触发退出禅模式
- **则** 布局恢复到进入前的侧栏/面板状态
- **并且** 不丢失用户原始布局偏好

### Requirement: 禅模式切换必须复用既有布局状态契约 [MODIFIED]

禅模式切换行为必须与现有布局状态管理与快捷键体系保持一致，不引入平行状态源。

#### Scenario: S3-ZEN-MODE-S3 快捷键切换与布局状态保持一致 [MODIFIED]

- **假设** 用户通过快捷键切换禅模式
- **当** 状态从普通模式切换到禅模式再切回
- **则** layout store 状态与 UI 展示一致
- **并且** 不出现双重切换或状态残留

## Out of Scope

- 编辑器内容模型与格式功能扩展
- 新主题或视觉系统改造
- 新增 IPC 或主进程能力
