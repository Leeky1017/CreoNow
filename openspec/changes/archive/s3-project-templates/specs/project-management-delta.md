# Project Management Specification Delta

## Change: s3-project-templates

### Requirement: 项目创建必须支持模板化初始化 [ADDED]

项目管理模块必须支持在创建项目时选择模板，并基于模板初始化项目结构与基础内容。

#### Scenario: S3-PROJECT-TPL-S1 选择内置模板创建项目 [ADDED]

- **假设** 用户在新建项目流程中选择内置模板（小说/短篇/剧本/自定义）
- **当** 用户确认创建
- **则** 系统按模板生成项目初始结构（基础文档/章节）
- **并且** 创建完成后可直接进入编辑流程

#### Scenario: S3-PROJECT-TPL-S2 模板应用后保持项目创建契约稳定 [MODIFIED]

- **假设** 前端仍通过既有 `project:create` 入口发起创建
- **当** 请求包含模板参数
- **则** 对外响应结构与错误语义保持兼容
- **并且** 模板仅影响初始化内容，不改变基础创建契约

### Requirement: 模板输入必须通过 schema 校验 [ADDED]

模板输入必须经过 schema 校验，非法自定义模板需被拒绝并返回明确错误。

#### Scenario: S3-PROJECT-TPL-S3 非法模板被拒绝并返回明确错误 [ADDED]

- **假设** 用户提交字段缺失或类型错误的自定义模板
- **当** 系统执行模板应用前校验
- **则** 创建流程被阻断
- **并且** 返回明确错误信息指示无效字段

## Out of Scope

- 在线模板市场与云端同步
- 模板版本管理与迁移系统
- AI 自动生成模板能力扩展
