# Editor Specification Delta

## Change: s2-shortcuts

### Requirement: 写作技能快捷键映射与触发分发 [ADDED]

系统必须提供编辑器快捷键映射与分发能力，用于触发写作技能入口，至少覆盖 `Ctrl+Enter`（续写）与 `Ctrl+Shift+R`（润色）。

- 快捷键映射必须集中定义在可测试的映射表中。
- 命中映射时必须路由到对应技能触发入口。
- 未命中映射时不得触发任何写作技能。

#### Scenario: Ctrl+Enter 触发续写技能 [ADDED]

- **假设** 编辑器处于可编辑状态且技能触发入口可用
- **当** 用户按下 `Ctrl+Enter`
- **则** 系统触发续写技能入口
- **并且** 不触发润色等非目标技能

#### Scenario: Ctrl+Shift+R 触发润色技能 [ADDED]

- **假设** 编辑器处于可编辑状态且技能触发入口可用
- **当** 用户按下 `Ctrl+Shift+R`
- **则** 系统触发润色技能入口
- **并且** 不触发续写等非目标技能

#### Scenario: 未命中快捷键组合不触发技能 [ADDED]

- **假设** 快捷键系统已加载映射
- **当** 用户输入未配置的组合键
- **则** 系统不触发任何写作技能
- **并且** 快捷键分发器保持空操作结果
